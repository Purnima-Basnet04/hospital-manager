Adding Doctor Images to Your Hospital Management System

Note: This is a simple guide to manually add doctor images to your project.

Step 1: Download Doctor Images

Download the following doctor images by right-clicking on each link and selecting "Save Link As...":

Dr. James Wilson (Chief Executive Officer) Dr. Sarah Johnson (Chief Medical Officer) Dr. Michael Chen (Director of Operations) Ms. Emily Rodriguez (Chief Nursing Officer) Generic Doctor Image (for placeholders)

Step 2: Create the Doctors Directory

Create a directory for doctor images in your project:

src/main/webapp/images/doctors/

Step 3: Save Images with Correct Names

Save the downloaded images with these exact filenames:

  • Dr. James Wilson → dr-james-wilson.jpg
  • Dr. Sarah Johnson → dr-sarah-johnson.jpg
  • Dr. Michael Chen → dr-michael-chen.jpg
  • Ms. Emily Rodriguez → ms-emily-rodriguez.jpg
  • Generic Doctor Image → doctor-placeholder.jpg

Step 4: Restart Your Server

After adding all images to the correct location, restart your application server to see the changes.

Alternative Solution: If you're unable to download the images, you can use placeholder images from a service like:

  • https://placehold.co/400x400?text=Doctor

In this case, update your JSP files to use these placeholder URLs instead.
